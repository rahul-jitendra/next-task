Trust Model Notes
• User key is the root of trust. No vendor CA exists.
• Hub certificate is user-signed. Hub cannot self-issue authority.
• Device certificates are hub-signed, user-authorized via lineage.
• All command capsules carry dual signatures (user + hub).
• ESP32 verifies signatures offline without hub contact.
• Nonce prevents replay attacks. Expiry prevents stale execution.
